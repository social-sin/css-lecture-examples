Advanced CSS Motion & Transforms

Learn how transitions, keyframes, transforms, and 3D motion create interactive and responsive interfaces while maintaining accessibility.

1. CSS Transitions

Transitions animate property changes between two states.

Hover Me 
transition:
transform 0.3s ease,
opacity 0.2s linear;

2. CSS Keyframe Animations

Keyframes define multi-step animation sequences.

Spin 
@keyframes spin {

  from {
    transform: rotate(0deg);
  }

  to {
    transform: rotate(360deg);
  }

}

10. Motion Accessibility

Respect user motion preferences with prefers-reduced-motion.

Reduced Motion Safe 
@media
(prefers-reduced-motion: reduce) {

  * {
    animation: none !important;
    transition: none !important;
  }

}

11. Performance Tips

  • Prefer transform + opacity for GPU acceleration
  • Avoid animating layout-heavy properties like width
  • Use subtle 3D transforms for realism
  • Avoid distracting infinite loops
  • Always support reduced motion
