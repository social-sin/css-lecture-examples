Combine transform and box-shadow for interactive depth.

box-shadow: 0 10px 20px rgba(...);

4. Animated Gradient Backgrounds

Animate gradients using background-position transitions.

5. Combined Keyframe Effects

Multiple transforms and opacity changes create richer motion.

Bounce 
@keyframes bounceFade {
  0% { opacity: 0; }
  50% { transform: translateY(-20px); }
}

6. Multiple Animations Together

Combine animations using commas.

Rotate + Pulse

pulse 1s ease,
rotate 4s linear infinite;

8. animation-fill-mode

animation-fill-mode controls whether animated styles persist before or after the animation finishes.

Default (none)

Snaps Back

forwards

Stays Final 
animation-fill-mode: forwards;

Without forwards, elements return to their original state.

9. Hover Triggered Keyframes

Hover interactions can trigger short animation sequences. 

.button:hover {
  animation: pulse 0.6s forwards;
}

10. CSS Variables for Animation Systems

CSS variables improve consistency and reusability.

Variable Animation 
:root {
  --anim-duration: 0.8s;
}
