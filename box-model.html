Box Sizing

Compare how total element size changes between content-box and border-box.

box-sizing: content-box

content-box

Declared width: 260px

Padding and border are ADDED to the declared width.

box-sizing: border-box

border-box

Declared width: 260px

Padding and border are INCLUDED inside the declared width. 

box-sizing: border-box;

Quick Reference

Property Defines Example
