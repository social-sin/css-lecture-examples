Advanced Responsive Images

Learn how responsive image techniques improve performance, quality, and adaptability across devices.

1. Standard Responsive Image Setup

Base responsive image CSS prevents overflow and preserves aspect ratio. 

img {
  max-width: 100%;
  height: auto;
  display: block;
}

2. Using srcset

The browser automatically chooses the best image size for the device. 

srcset="
image-400.jpg 400w,
image-800.jpg 800w,
image-1200.jpg 1200w
"

3. Adding sizes

sizes tells the browser how much screen space the image will occupy.

Text Content

On larger screens this layout becomes two columns, so the image only occupies about half the viewport width. 

sizes="
(max-width: 768px) 100vw,
50vw
"

4. <picture> Element (Art Direction)

picture allows different images for different screen sizes. 

<picture>
  <source media="(max-width: 768px)">
  <source media="(min-width: 769px)">
  <img>
</picture>

5. Format Switching

Browsers can load optimized image formats like AVIF and WebP automatically. 

<source
  srcset="image.webp"
  type="image/webp"
>

6. Lazy Loading

Images load only when approaching the viewport for better performance. 

loading="lazy"

7. Combining Techniques

Responsive resolution, adaptive sizing, and lazy loading can work together. 

srcset + sizes + loading="lazy"

8. Why Responsive Images Matter

Responsive images improve loading speed, reduce bandwidth usage, and create better mobile experiences.

Unoptimized Large Image
