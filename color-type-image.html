CSS Color, Type, and Images

Learn how CSS controls color systems, typography styling, filters, and background images.

Colors

CSS supports multiple color formats.

Hexadecimal

Same color • different syntax

#ff6600

RGB

Same color • different syntax

rgb(255, 102, 0)

RGBA

Includes transparency

rgba(255, 102, 0, 0.8)

HSL

Hue • Saturation • Lightness

hsl(24, 100%, 50%)

HSLA

HSL with transparency

hsla(24, 100%, 50%, 0.7)

Pro Tip: Use rgba() or hsla() for overlays and transparency effects.

Color Contrast Accessibility

Strong contrast improves readability and accessibility.

Good Contrast

This text is easy to read because the foreground and background colors have strong contrast.

Bad Contrast

This text is difficult to read because the colors are too similar.

Accessibility Tip

WCAG recommends a contrast ratio of at least 4.5:1 for normal body text.

line-height

Tight line-height makes paragraphs feel compact and more difficult to scan.

Larger line-height improves readability and creates breathing room between lines. 

line-height: 2;

Small text

Large text

Background Images

Background images can be positioned, scaled, and layered with overlays.

background-size: cover;
background-size: contain;
background-attachment: fixed; 
background-image: url("photo.jpg");
background-size: cover;
background-position: center;

Shorthand: background: url("photo.jpg") no-repeat center/cover;

CSS Filter Effects

CSS filters apply visual effects to images.

filter: blur() 

filter: blur(6px);

filter: grayscale() 

filter: grayscale(100%);

filter: brightness() 

filter: brightness(160%);
