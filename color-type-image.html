CSS Color, Type, and Images

Learn how CSS controls color systems, typography styling, and background images.

Colors

CSS supports multiple color formats.

Hexadecimal

color: #ff6600;

RGB

color: rgb(255, 102, 0);

RGBA

rgba(255, 102, 0, 0.8);

HSL

hsl(24, 100%, 50%);

HSLA

hsla(24, 100%, 50%, 0.7);

Pro Tip: Use rgba() or hsla() for transparent overlays.

Color Contrast Accessibility

Strong contrast improves readability and accessibility, especially for users with visual impairments.

Good Contrast

This text is easy to read because the foreground and background colors have strong contrast.

Bad Contrast

This text is difficult to read because the colors are too similar in brightness and tone.

Accessibility Tip

Use contrast checking tools to ensure readable text, especially for smaller typography and interactive elements.

WCAG recommends a contrast ratio of at least 4.5:1 for normal body text.

Background Images

Background images can be positioned, scaled, and layered with overlays.

background-size: cover;
background-size: contain;
background-attachment: fixed; 
background-image: url("photo.jpg");
background-repeat: no-repeat;
background-size: cover;
background-position: center;

Shorthand: background: url("photo.jpg") no-repeat center/cover;

CSS Filter Effects

CSS filters apply visual effects to images like blur, grayscale, and brightness adjustments.

filter: blur()

filter: blur(6px);

filter: grayscale()

filter: grayscale(100%);

filter: brightness()

filter: brightness(160%);
