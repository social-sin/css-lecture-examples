CSS Color, Type, and Images

Learn how CSS controls color systems, typography styling, and background images.

Colors

CSS supports multiple color formats.

Hexadecimal

color: #ff6600;

RGB

color: rgb(255, 102, 0);

RGBA

rgba(255, 102, 0, 0.8);

HSL

hsl(24, 100%, 50%);

HSLA

hsla(24, 100%, 50%, 0.7);

Pro Tip: Use rgba() or hsla() for transparent overlays.

Background Images

Background images can be positioned, scaled, and layered with overlays.

background-size: cover;
background-size: contain;
background-attachment: fixed; 
background-image: url("photo.jpg");
background-repeat: no-repeat;
background-size: cover;
background-position: center;

Shorthand: background: url("photo.jpg") no-repeat center/cover;

Quick Summary

Category Common Properties Purpose
Colors hex, rgb(), hsl() Create visual identity
Typography font-size, line-height Improve readability
Images background-size, position Control image scaling
