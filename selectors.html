CSS Selectors Cheatsheet

Learn how selectors target elements using classes, IDs, combinators, pseudo-classes, and document-level foundations.

1. Basic Selectors

Root Selector (:root)

The :root selector targets the highest-level element in the document.

In HTML documents, :root refers to the <html> element.

Developers commonly use :root to store global CSS variables. 

:root {

  --primary-color: #3498db;
  --spacing: 20px;

}

Universal Selector (*)

✓ Affects ALL elements

The universal selector targets every element on the page.

  • box-sizing
  • font-family
  • margin reset

html Selector

The html selector targets the root element of the webpage.

  • Root font sizing
  • Smooth scrolling
  • Theme variables
  • Global styling foundations
html {
  font-size: 16px;
  scroll-behavior: smooth;
}

The html element is the root of the DOM tree.

Element Selector

✓ Affected by: p

All paragraph elements are styled using the p selector.

Every paragraph shares the same text color and spacing.

Class Selector (.class)

This paragraph has the class "highlight".

Only elements with the class receive the highlighted style.

ID Selector (#id)

✓ Example: #header

IDs should be unique per page.

ID selectors are highly specific.
