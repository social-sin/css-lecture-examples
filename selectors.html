CSS Selectors Cheatsheet

This page demonstrates basic selectors, combinators, and link pseudo-classes.

1. Basic Selectors

Universal Selector (*)

The universal selector targets every element on the page.

In this example, the universal selector applies:

  • box-sizing
  • font-family
  • margin reset

html Selector

The html selector targets the root element of the entire webpage.

Because every element exists inside the html element, it is commonly used for:

  • Root font sizing
  • Smooth scrolling
  • Global theme variables
  • Full-page styling foundations

Many responsive websites define base typography on the html element using rem units. 

html {
  font-size: 16px;
  scroll-behavior: smooth;
}

The html element is the root of the DOM tree.

Element Selector

All paragraph elements are styled using the p selector.

Every paragraph shares the same text color and spacing.

Class Selector (.class)

This paragraph has the class "highlight".

Only elements with the class receive the highlighted style.

ID Selector (#id)

The page header uses the ID selector: #header

IDs should be unique per page.
