CSS Units & Values

CSS units control sizing, spacing, typography, and layout behavior.

Unit Description Example
px Fixed pixel size width: 200px;
% Relative to parent width: 50%;
em Relative to parent font size padding: 2em;
rem Relative to root font size font-size: 1.2rem;
vh Viewport height height: 100vh;
vw Viewport width width: 100vw;

Modern Responsive Typography with clamp()

Modern responsive typography often uses clamp() to create scalable text with minimum and maximum size limits.

Responsive clamp() Heading

Resize the browser window to see how the text smoothly scales between a minimum and maximum size.

Viewport Units Only

font-size: 3vw;

Can become too small on mobile or too large on huge screens.

clamp()

clamp(2rem, 5vw, 5rem)

Sets a minimum, preferred, and maximum responsive size. 

font-size: clamp(2rem, 5vw, 5rem);

/*
Minimum Size: 2rem
Preferred Scaling: 5vw
Maximum Size: 5rem
*/

clamp() is now one of the most common techniques for responsive typography.
