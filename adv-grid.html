Advanced CSS Grid

Build powerful, semantic, and responsive layouts using advanced CSS Grid features.

1. grid-template-areas + grid-area

Named grid areas create readable, visual layout structures.

Header
Sidebar
Main Content
Footer 
grid-template-areas:
  "header header"
  "sidebar main"
  "footer footer";

Each string represents one grid row.

2. Fractional Units (fr)

The fr unit divides available space proportionally.

1fr
2fr
1fr
grid-template-columns:
1fr 2fr 1fr;

3. minmax()

Flexible track sizing that adapts between minimum and maximum values.

Card 1
Card 2
Card 3
grid-template-columns:
repeat(3, minmax(200px, 1fr));

4. auto-fill

Creates as many columns as fit, even if some tracks remain empty.

1
2
3
4
grid-template-columns:
repeat(auto-fill, minmax(180px, 1fr));

5. auto-fit

Empty columns collapse automatically, creating more responsive layouts.

B
C
D
grid-template-columns:
repeat(auto-fit, minmax(180px, 1fr));

7. Empty Grid Spaces (.)

Use a period (.) to create empty grid cells.

Header
Sidebar
Main 
grid-template-areas:
"header header"
"sidebar .";
