Advanced CSS Grid

Build powerful, semantic, and responsive layouts using advanced CSS Grid features.

1. grid-template-areas + grid-area

Named grid areas create readable, visual layout structures.

Header
Sidebar
Main Content
Footer 
grid-template-areas:
  "header header"
  "sidebar main"
  "footer footer";

Each string represents one grid row.

2. Fractional Units (fr)

The fr unit divides available space proportionally.

1fr
2fr
1fr
grid-template-columns:
1fr 2fr 1fr;

3. minmax()

Flexible track sizing that adapts between minimum and maximum values.

Card 1
Card 2
Card 3
grid-template-columns:
repeat(3, minmax(200px, 1fr));

4. auto-fill

auto-fill creates as many columns as possible, even if some tracks remain empty.

Resize the browser window to compare how auto-fill behaves differently from auto-fit.

1
2
3
 grid-template-columns: repeat(auto-fill, minmax(180px, 1fr));

5. auto-fit

auto-fit collapses empty tracks, allowing existing items to stretch.

Resize the browser window and compare this layout against auto-fill.

B
C

9. Empty Grid Spaces (.)

Use a period (.) to create empty grid cells.

Header
Sidebar
Main

"header header"
"sidebar .";

10. Explicit vs Implicit Grid

Explicit grids are manually defined using grid-template-columns and grid-template-rows.

Implicit rows are automatically created when extra items overflow the layout.

1
2
3
4
5
6
 grid-template-columns: repeat(3, 1fr); grid-auto-rows: 100px;

Implicit tracks are created automatically when content exceeds the explicit grid.
