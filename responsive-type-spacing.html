Responsive Images, Typography & Spacing

Learn how media, text, and layout spacing adapt gracefully across different screen sizes.

Responsive Images & Media

Images and videos should scale fluidly inside their containers. 

img,
video {
  max-width: 100%;
  height: auto;
  display: block;
}

object-fit: cover

object-fit helps media fill containers without distorting aspect ratio. 

img {
  object-fit: cover;
}

Responsive Typography

Typography can scale dynamically using viewport units and relative sizing.

Fluid Heading

This paragraph uses rem units and line-height to maintain readability across devices. 

h1 {
  font-size: 3vw;
}

p {
  font-size: 1rem;
  line-height: 1.5;
}

Fluid Typography with clamp()

clamp() scales text responsively while keeping readable limits.

Resize the Browser Window 

font-size:
clamp(1.5rem, 4vw, 3rem);

Responsive Grid Layout

Grid layouts can automatically reflow across screen sizes.

Flexible Grid Item

Grid layouts adapt automatically with auto-fit and minmax().

Scalable Layout

Images, typography, and spacing scale together responsively.

Mobile Friendly

Cards wrap automatically as the viewport shrinks. 

grid-template-columns:
repeat(auto-fit, minmax(250px, 1fr));

Media Queries

Media queries apply styles at specific screen widths.

Resize the browser window 
@media (max-width: 768px) {
  h1 {
    font-size: 5vw;
  }
}

SVG Graphics

SVGs scale infinitely without losing quality. 

svg {
  width: 100%;
  height: auto;
}

Combining Techniques

Fully responsive sections combine:
