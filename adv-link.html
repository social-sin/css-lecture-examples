Basic Styled Links

Links should remain semantic while gaining visual clarity and interactivity.

Standard Interactive Link 
a {
  text-decoration: none;
  transition: 0.3s ease;
}

Sliding Underline Effect

Use pseudo-elements and transitions to animate underlines smoothly.

Hover Me 
a::after {
  width: 0;
  transition: width 0.3s ease;
}

a:hover::after {
  width: 100%;
}

Underline Growing from Center

Center Animation 
left: 50%;

a:hover::after {
  width: 100%;
  left: 0;
}

Accessible Focus States

Keyboard users should clearly see which element is focused.

Press TAB to Focus 
a:focus-visible {
  outline: 2px solid #2ecc71;
}

External Link Indicators

Indicate when a link opens in a new tab.

External Resource

  content: " ↗";
}
